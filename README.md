# images

お知らせに使用する画像ファイルをこのフォルダに置いてください。

## 画像のURL形式

画像をアップロードしたら、以下のURLをGoogleスプレッドシートの `imageURL` 列に入力します。

```
https://raw.githubusercontent.com/zeromiss-official/ZeroMiss/main/images/ファイル名.png
```

例：`notice_2026_03.png` という画像をアップロードした場合
```
https://raw.githubusercontent.com/zeromiss-official/ZeroMiss/main/images/notice_2026_03.png
```
